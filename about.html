About This Project

This web site was created by Cole and Jen Jayme for a 6th grade school research project. Cole is a martial artist and tricker with an interest in science. His mom Jen is a web developer. Cole came up with the idea, researched the topics, and found pictures and videos. Jen helped him put these together on a web site. Hope you enjoy!
